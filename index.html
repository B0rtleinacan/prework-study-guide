HTML

  • head: element contains information about the webpage.
  • body: element represents the visible content shown to the User.
  • charset: sets character encoding.
  • http-equiv: sets an HTTP header for content.
  • style: defines the CSS style associated with a website.
  • viewport: sets the default view for out webpage.
  • link: connects an external resource to the HTML document.
  • meta: defines the metadata such as the character set, description, keywords, author and viewport.
  • base: defines the base URL for a webpage.

CSS

  • margin: indicates how much space we want around the outside of an element.
  • padding: indicates how much space we want around the content inside an element.
  • box-shadow: CSS property adds shadow effects around an element's frame.
  • Class Attributes: allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • The CSS box model as a whole applies to block boxes and defines how the different parts of a box — margin, border, padding, and content — work together to create a box that you can see on a page. Inline boxes use just some of the behavior defined in the box model.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Adds changes to branch.
  • git pull origin main: recieves a branch's modification into the local branch.
  • git push origin main: add's new changes to an existing branch.
  • git checkout --: moves the working branch to a new branch.
  • git commit -m "": commits changes to branch.
  • code . : opens VS Code.
  • cd: change directory
  • cd .. : allows back tracking in directory
  • ls: shows list of files in directory
  • pwd: shows location in directory
  • start: opens file in directory
  • touch: creates file in directory
  • mkdir: creates new directory

JavaScript

  • var: container for variables before 'let' and 'const' became a thing
  • Function: something to call with block of code that executes a specific task
  • For loop: allows specific things to be executed until the condition is not met.
  • Arrays: a single variable that is used to hold a group of data. Typically written with [] after a variable has been declared.
